О нас

Наша компания является производителем товаров хозяйственно-бытового назначения.

Уже больше 1000 лет наше производство предлагает широкий ассортимент продукции:

  • -Мешки для мусора (ПНД, ПВД)
  • -Фасовочные пакеты
  • -Губки для посуды
  • -Тряпки для пола
  • -Салфетки для стола (хлопок, вискоза, микрофибра)

Ознакомиться более подробно с нашим предложением вы можете в разделе Каталог.

FAQ

Через сайт:

  1. Выберите товар и нужное количество
  2. По окончании выбора товаров перейдите в корзину. В ней находятся все выбранные товарные позиции. Их количество можно увеличить или уменьшить.
  3. Нажмите кнопку "Оформить заказ" и переходите к заполнению формы.
  4. После заполнения формы, менеджер, обрабатывающий ваш заказ, свяжется с вами для уточнения всех деталей.
  5. Для оплаты заказа по безналичному расчёту вам будет выслан счет.

По телефону:

  1. Позвоните по номеру +7 (812) 448-59-97 и озвучьте ваш заказ менеджеру.
  2. Менеджер предоставит вам всю необходимую информацию по телефону.
  3. Для оплаты заказа по безналичному расчёту вам будет выслан счет.

По e-mail:

  1. Отправьте нам письмо на alpak@alpak.spb.ru с перечнем товаров и комментариями к заказу.
  2. Менеджер, обрабатывающий ваш заказ, свяжется с вами для уточнения всех деталей.
  3. Для оплаты заказа по безналичному расчёту вам будет выслан счет.

Наличие продукции

Непрерывное производство позволяет поддерживать складские остатки в необходимом минимуме и при необходимости достичь нужного количества по заказу клиента в минимальный срок.

02

Политика цен

В зависимости от количества выбранного товара и формы оплаты возможны скидки или цена по договоренности.

03

Отсрочка платежа

Предоставляем отсрочку платежа нашим постоянным клиентам до 60 дней.

04

Удобная логистика

При заказе от 15.000 рублей доставка в черте города Санкт-Петербурга предоставляется бесплатно. Доставка в другие регионы страны осуществляется транспортными компаниями

05

Изготовление продукции на заказ под вашим брендом

Изготовим продукцию по размерам заказчика "под ключ". Дополнительные услуги включают в себя разработку индивидуальной этикетки и упаковки для заказчика.

Возможные варианты поставки товара:

• с этикеткой / без этикетки / с этикеткой заказчика;

• в упаковке / без упаковки / в упаковке заказчика.
